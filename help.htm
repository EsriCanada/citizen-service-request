Citizen Service Request

Online Help

The Citizen Service Request application allows citizens to submit requests for service in their municipality. It is a single application that can be used on a desktop browser, mobile phone, or tablet device. This application provides 24x7 access to the municipality and typically supplements customer service phone numbers.

Address Search

  1. If necessary, select the “Address” radio button.
  2. Enter an address in the search bar.
  3. Press enter to search immediately or select a result from the autocomplete list.
Those using a mobile phone or older browsers will need to select the Search icon
to display the search field.

My Location Search

The user can select the Geolocation icon
to conduct a search based on their current location.

Creating a New Service Request

After finding the location where a new service request is to be created:

  1. Click or tap the map at the location that requires a new service request.
  2. Fill out the details popup with the required information. For contact information, either a phone number or an email address is required.
  3. You may optionally add an picture to the request.
Once submitted, you will see a new red point on the map indicating that your service request has been received but has yet to be evaluated. If you wish to keep track of your request, the request ID found at the top of the popup window will allow you to pull-up the request at a later date.

Finding an Existing Request by Request ID

If you know the Request ID of an existing service request:

  1. If necessary, select the “Request ID” radio button.
  2. Enter the Request ID in the search field.
  3. Press enter or click the search button.
Those using a mobile phone or older browsers will need to select the Search icon
to display the search field.

Viewing & Commenting on Existing Service Requests

You may click an existing service request on the map to display its details. By clicking the Comment
button in the details popup, you may also add a comment to the request.

Map Functions

  • Use the basemap selector
    (not available on mobiles) to display your preferred basemap.
  • Select + or – in the upper left of the map to zoom in or out.

Share Your Map

You can click on the Share icon
to share your map in an email, a Facebook post or on Twitter.

Browser Compatibility

The My Municipal Government Services app is supported on the following browsers: Microsoft Internet Explorer 7/8/9/10, Mozilla Firefox, Google Chrome, Apple Safari, iOS Safari 4.0+, Android Chrome 2.3+, and most modern JavaScript enabled browsers.
